## 專案概況

**蛙抵家（Identifrog）** 是一款以 AI 聲紋辨識為核心的公民科學行動應用程式，協助民眾記錄臺灣蛙類分布資料。2.0 版本在現有基礎上擴展為三層應用場景：

| 功能層 | 目標使用者 | 核心新增功能 |
|--------|-----------|-------------|
| 公民科學層 | 自然愛好者、一般民眾 | 辨識結果不確定性設計、蛙鳴地圖升級 |
| 智慧導覽層 | 環境教育場域、學生 | 場域聲景體驗、倫理聆聽協議 |
| 專業監測層 | 環境顧問公司、研究者 | PAM 資料匯入、批次分析、報告產出 |

---

## 設計哲學：環境感知 UI（Eco-Aware UI）

本專案採用「環境感知 UI」作為核心設計語言，四個核心原則：

1. **不確定性是資訊，不是失敗** — 辨識信心程度應清楚呈現，而非隱藏
2. **科技為關係服務** — 引導用戶注意當下環境，而非只盯著螢幕
3. **倫理聆聽** — 季節提醒、減光建議、靜音區域等對環境的責任感設計
4. **溫暖科技** — 柔和自然的視覺語言，強調人與自然的關係性連結
